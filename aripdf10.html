Page 1: Introduction

Arithmetic Progression (AP): sequence where each term after first is obtained by adding constant difference to previous term.

Constant difference = common difference (d).

Example: 2, 5, 8, 11,... (d=3)

Page 2: General Form of AP

a, a+d, a+d+d, ... or a, a+d, a+2d, ...

a = first term

d = common difference

If d > 0 ‚Üí increasing AP
d < 0 ‚Üí decreasing AP
d = 0 ‚Üí constant sequence

Page 3: nth Term of AP

a_n = a + (n-1)d

n = term number

Most important formula.

Page 4: Sum of First n Terms

S_n = n/2 [2a + (n-1)d]
or S_n = n/2 (a + l) (l = last term)

Derived from pairing terms.

Page 5: Properties of AP

  • If same term added/subtracted ‚Üí still AP
  • If multiplied/divided by non-zero constant ‚Üí still AP
  • Three terms a-d, a, a+d ‚Üí symmetric

Page 6: Selecting Terms in AP

For 3 terms: a-d, a, a+d

For 4 terms: a-3d, a-d, a+d, a+3d

For 5 terms: a-2d, a-d, a, a+d, a+2d

Page 7: Word Problems

Form AP from statement ‚Üí use a_n or S_n.

Common: savings, salary, seats, wages.

Page 8: Key Formulas Summary

a_n = a + (n-1)d
S_n = n/2 [2a + (n-1)d]
S_n = n/2 (a + l)
