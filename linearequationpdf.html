Page 1: Introduction to Linear Equations

A linear equation in two variables x and y is of the form ax + by + c = 0, where a, b, c are real numbers and a, b not both zero.

Examples: 2x + 3y = 12, x - y + 5 = 0, 4x = 8 (here b=0).

Every solution (x, y) satisfying the equation represents a point on the line.

This chapter teaches how to solve, graph, and apply such equations.

Page 2: Solution of a Linear Equation

Any pair (x, y) that satisfies ax + by + c = 0 is a solution.

There are infinitely many solutions (corresponding to points on the line).

Example: For 2x + y = 5
If x=0 ‚Üí y=5 (point (0,5))
If x=1 ‚Üí y=3 (point (1,3))
If x=2 ‚Üí y=1 (point (2,1))

Page 3: Graph of a Linear Equation

The graph of ax + by + c = 0 is always a straight line.

To plot: Find at least two points ‚Üí join them.

Intercepts:

  • x-intercept: Put y=0 ‚Üí solve for x
  • y-intercept: Put x=0 ‚Üí solve for y

Plotting these two intercepts is the fastest method.

Page 4: Examples of Graphing

Example 1: Draw 2x + 3y = 12
x-intercept: y=0 ‚Üí 2x=12 ‚Üí x=6 (6,0)
y-intercept: x=0 ‚Üí 3y=12 ‚Üí y=4 (0,4)
Plot (6,0) and (0,4) ‚Üí straight line.
Example 2: x + y = 0 ‚Üí passes through origin.

Page 5: Equations of Lines Parallel to Axes

x = a ‚Üí vertical line (parallel to y-axis)

y = b ‚Üí horizontal line (parallel to x-axis)

Example: x = 3 ‚Üí all points (3, y)
y = -2 ‚Üí all points (x, -2)

No intercepts needed ‚Äî just draw accordingly.

Page 6: System of Two Linear Equations

Two equations together: a‚ÇÅx + b‚ÇÅy + c‚ÇÅ = 0 and a‚ÇÇx + b‚ÇÇy + c‚ÇÇ = 0

Three possible cases:

  • Intersecting lines ‚Üí unique solution (consistent)
  • Parallel lines ‚Üí no solution (inconsistent)
  • Coincident lines ‚Üí infinitely many solutions (dependent)

Page 7: Graphical Method of Solution

Plot both lines.

  • Intersection point ‚Üí solution
  • No intersection ‚Üí no solution
  • Lines overlap ‚Üí infinite solutions
Example: Solve x + y = 5 and x - y = 1 graphically.
Intersection at (3,2).

Page 8: Algebraic Methods - Substitution

Steps:

  1. Express one variable in terms of other from one equation.
  2. Substitute into second equation.
  3. Solve for one variable.
  4. Find second variable.
Example: x + y = 7, x - y = 3
x = 7 - y ‚Üí substitute ‚Üí (7-y) - y = 3 ‚Üí x=5, y=2

Page 9: Algebraic Methods - Elimination

Make coefficients of one variable equal ‚Üí add/subtract.

Example: 2x + 3y = 12, 4x - y = 5
Multiply second by 3: 12x - 3y = 15
Add: 14x = 27 ‚Üí x=27/14, then y.

Page 10: Cross-Multiplication Method

For a‚ÇÅx + b‚ÇÅy + c‚ÇÅ = 0 and a‚ÇÇx + b‚ÇÇy + c‚ÇÇ = 0
x = (b‚ÇÅc‚ÇÇ - b‚ÇÇc‚ÇÅ)/(a‚ÇÅb‚ÇÇ - a‚ÇÇb‚ÇÅ)
y = (c‚ÇÅa‚ÇÇ - c‚ÇÇa‚ÇÅ)/(a‚ÇÅb‚ÇÇ - a‚ÇÇb‚ÇÅ)

Denominator zero ‚Üí parallel/infinite cases.

Page 11: Consistency Conditions

  • a‚ÇÅ/a‚ÇÇ ‚â† b‚ÇÅ/b‚ÇÇ ‚Üí unique solution
  • a‚ÇÅ/a‚ÇÇ = b‚ÇÅ/b‚ÇÇ ‚â† c‚ÇÅ/c‚ÇÇ ‚Üí no solution
  • a‚ÇÅ/a‚ÇÇ = b‚ÇÅ/b‚ÇÇ = c‚ÇÅ/c‚ÇÇ ‚Üí infinite solutions

Memorise these ratios!

Page 12: Key Formulas Summary

  • General form: ax + by + c = 0
  • x-intercept = -c/a, y-intercept = -c/b
  • Parallel axes: x=a, y=b
  • Consistency ratios
  • Cross-multiplication formula

  2. Find intercepts of 3x + 4y = 12.
  3. Draw x = 4.
  4. Solve x + y = 5, x = 3.
  5. Is (2, -1) solution of 3x - 2y = 8?
  6. Express y = 5 - 2x in ax + by + c = 0.
  7. Find one solution of x - y = 0.
  8. Graph y = 0.

Page 14: Practice Questions - Medium (11-20)

  1. Solve graphically: 2x + y = 6, 4x + 2y = 12.
  2. Solve by substitution: 3x - y = 3, 9x - 3y = 9.
  3. Find k for no solution: 2x + ky = 8, 3x + 2y = 12.
  4. Solve: x/2 + y/3 = 4, x/4 + y/6 = 3.
  5. Graph x + 2y = 6 and find area with axes.
  6. Solve elimination: 5x - 3y = 1, 2x + y = 7.
  7. Check consistency: 4x + 6y = 12, 2x + 3y = 6.
  8. Find equation of line through (0,3) and (4,0).
  9. Solve cross-multiplication: 3x + 4y = 10, 2x - y = 5.
  10. Word problem setup: Sum of ages.
