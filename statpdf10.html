Page 1: Introduction

Statistics deals with collection, organisation, analysis of data.

Measures of central tendency: mean, median, mode.

For grouped data: methods and graphical representation.

Page 2: Mean - Direct Method

Mean = Œ£(f_i x_i) / Œ£f_i

x_i = class mark, f_i = frequency

Page 3: Mean - Assumed Mean Method

Mean = a + Œ£(f_i d_i) / Œ£f_i
d_i = x_i - a

a = assumed mean (usually middle class mark)

Page 4: Mean - Step Deviation Method

Mean = a + h √ó Œ£(f_i u_i) / Œ£f_i
u_i = (x_i - a)/h

h = class size (when uniform)

Page 5: Median

Median = l + [(N/2 - cf)/f] √ó h

l = lower limit of median class
cf = cumulative frequency before median class
f = frequency of median class
N = Œ£f

Page 6: Mode

Mode = l + [(f_1 - f_0)/(2f_1 - f_0 - f_2)] √ó h

f_1 = modal class frequency
f_0 = previous, f_2 = next

Page 7: Cumulative Frequency Curve (Ogive)

Less than ogive: plot upper limit vs cumulative frequency

More than ogive: lower limit vs (N - cf)

Median from intersection at N/2

From histogram: highest bar ‚Üí modal class ‚Üí mode formula
